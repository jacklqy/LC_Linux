VSCode安装及.NET运行环境配置和测试

特别重要

在Centos7安装的VSCode版本必须是小于1.53.0的，建议安装1.52.1版本 亲测没有问题

1、官网下载VSCode安装包

wget https://vscode.cdn.azure.cn/stable/ea3859d4ba2f3e577a159bc91e3074c5d85c0523/code-1.52.1-1608137084.el7.x86_64.rpm

2、解压安装包到 /usr/local目录【必须以管理身份运行】

rpm -ivh code-1.52.1-1608137084.el7.x86_64.rpm

问题: libXss.so.1()(64bit) is needed by code-1.52.1-1608137084.el7.x86_64

在本地打开命令行：

wget http://mirror.centos.org/centos/7/os/x86_64/Packages/libXScrnSaver-1.2.2-6.1.el7.x86_64.rpm
下载成功后进行安装：

sudo rpm -ivh libXScrnSaver-1.2.2-6.1.el7.x86_64.rpm

3、在终端直接运行code

code

You are trying to start Visual Studio Code as a super user which isn’t recommended. If this was intended, please specify an alternate user data directory using the --user-data-dir argument.

原因：使用超级用户安装了 没有生成对应用户数据存储目录

解决： 在启动的时候指定对应–usedr-data-dir参数的值就OK

code --user-data-dir=~/.vscode

看到如下画面代表成功了

4、安装.NET的运行环境

文档DOC

https://docs.microsoft.com/en-us/dotnet/core/install/linux-centos

  1. 添加源

~shell
rpm -Uvh https://packages.microsoft.com/config/centos/7/packages-microsoft-prod.rpm
~

  1. 安装SDK

~shell
yum install dotnet-sdk-5.0
~

  1. 安装运行时

~shell
yum install aspnetcore-runtime-5.0
~

  1. 测试运行时是否成功

~shell
dotnet –version
5.0.301
~

5、创建.NET Core控制台项目并运行

dotnet new console --name DotNetCoreSample
### 定位到项目根目录执行
dotnet run
############ 输出内容 ############

6、各种命令操作很烦，安装Visual Studio Code中vscode-solution-explorer解决方案插件

7、添加Nuget依赖管理插件NuGet Package Manager

出现：Nuget插件版本问题修改修改文件：

/home/gerry/.vscode/extensions/jmrog.vscode-nuget-package-manager-1.1.6/out/src/actions/add-methods/fetchPackageVersions.js对应的内容修改为：

node_fetch_1.default(`${versionsUrl}${selectedPackageName}/index.json`.toLowerCase(), utils_1.getFetchOptions(vscode.workspace.getConfiguration('http')))

——————————————————— 输入依赖模糊搜索识别慢——————————————————–

修改配置文件：vi ~/.nuget/NuGet/NuGet.Config

<?xml version="1.0" encoding="utf-8"?>
<configuration>
  <packageSources>
    <add key="nuget.org" value="https://nuget.cnblogs.com/v3/index.json" protocolVersion="3" />
  </packageSources>
</configuration>
